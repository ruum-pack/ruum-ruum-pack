# Escala tipografica operativa

La interfaz de conductor usa una escala comun para conservar legibilidad en movil y zoom al 200%.

| Uso | Clase minima | Tamano |
| --- | --- | --- |
| Campos, selects y textareas | `text-base` | 16 px |
| Direcciones y destino actual | `text-base` | 16 px |
| Instrucciones operativas | `text-sm` | 14 px |
| Navegacion inferior | `text-xs` | 12 px |
| Badges y metadatos no criticos | `text-xs` | 12 px |

Reglas:

- Ninguna instruccion operativa debe bajar de `text-sm`.
- Las direcciones visibles durante un traslado usan minimo `text-base`.
- La fuente monoespaciada se reserva para IDs, folios, placas y montos.
- Evitar mayusculas sostenidas en instrucciones, ayudas, direcciones y controles; usarlas solo en etiquetas cortas de metadatos.
